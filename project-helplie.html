Helplie iOS & Android

Discount is a cross-platform mobile application designed to help users discover relevant deals based on their location and personal preferences. The platform aggregates discounts from multiple sources and presents them in a simple and user-friendly interface.

Personalized Deals & Discovery

The application focuses on delivering relevant offers to users by combining location-based data with user preferences. This allows users to easily discover discounts that are meaningful to them without manually searching across different platforms.

Authentication & User Profiles

Firebase Authentication was used to manage user accounts, enabling secure login and personalized experiences. Each user can maintain preferences and interact with content tailored specifically to their interests.

Affiliate Links & Monetization

Discounts are connected to affiliate links, allowing users to access offers directly while enabling potential monetization through referral-based tracking. This creates a bridge between users and businesses while supporting a scalable business model.

Real-Time Data with Firebase

Firebase was used as the backend for storing and updating discount data in real time. This ensures that users always see the most up-to-date offers and allows for easy management of content across platforms.

Cross-Platform Development

The app was developed for both iOS and Android using Swift and Kotlin, sharing a common backend and data structure. This approach ensured consistency across platforms while maintaining native performance and user experience.

Tech Stack

  • Swift (iOS)
  • Kotlin (Android)
  • Firebase Authentication
  • Firebase Realtime Database / Firestore
  • Affiliate link integration
